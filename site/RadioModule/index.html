    RadioModule

    class RadioModule

    Represents the long-range radio module (LoRa)

    Functions

    Name Description
    RadioModule The long-range radio module (LoRa) will use these pins for recieving and transmitting data rxPin : The pin that will be used for reciving data txPin : The pin that will be used for transmitting data
    begin Sets up the hardware UART connection and configures the radio module to upload mode
    sendPrimaryData Sends the primary data (temperature and pressure) to the ground station temperature : Temperature in degrees celsius pressure : Relative air pressure in kPa
    sendSecondaryData Sends the GPS position of the satellite to the ground station position : The current GPS position of the satellite
    setRadioFrequency Sets the operating radio frequency of the module radioFrequency : Radio frequency in kHz
